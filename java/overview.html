ALIX, A Lucene Indexer for XML documents

Alix is a java library to index and search XML text documents with Lucene including linguistic expertness for French, available under Apache licence.

Alix has been started in 2009 under the javacrim project for a java course at Inalco, given by Pierre Dittgen and Frédéric Glorieux.

Alix continues the concepts of SDX (« Système de Documentation en XML »), a project on which P. Dittgen and F. Glorieux were contributors.
